# Web_Semester04 #
## Dönerverleih ##
* Projekt aus dem Pratikum _Webbasierte Anwendungen_ (4. Semester) in Kooperation mit einem Kommolitonen
* Ziel: Vermittlung der Grundlagen von Frontend- und Backend-Entwicklung
* Backend entwickelt mit **Spring Boot** 
* Frontend umgesetzt mit **Vue.js**
* Kommunikation zwischen Frontend und Backend über **REST-API**
* Loginfunktion mit festen Zugangsdaten (Username: `admin` Password:`admin`), zur Vereinfachung gehardcoded (nur zur Demonstrationszwecken)
### Startanleitung ###
1. Backend starten  
`./gradlew bootRun`
2. Frontend starten
   * in Frontend-Ordner navigieren `cd frontend`
   * Frontend starten `npm run dev`
   
### Zielumgebung ###
* Backend standardmäßig auf `http://localhost:8080/doener`
* Frontend auf `http://localhost:5173/`
